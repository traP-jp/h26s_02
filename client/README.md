# h26s_02 frontend

### 環境構築

フロントエンドを触る場合、VSCode では **client ディレクトリを開く** ことをおすすめします！ ソースコードの Lint や補完にエラーが出づらくなるためです

リポジトリのディレクトリをシェルで開いて、以下を1行ずつ実行してください

```sh
cd client
corepack enable pnpm
pnpm install  # 必要なパッケージをインストール
```

開発環境を

```sh
pnpm dev
```

で立ち上げたあと、<https://localhost:5173> にアクセスすることで閲覧できます（http **s** です！）。これと VSCode のソースを交互に見ながら開発を進めます

### 実機確認

1. スマホのテザリングに開発中の PC を接続
2. `pnpm dev` を実行して `Network:` に続いて表示されている URL をスマホで開く
3. 証明書エラーで警告が出るけど「詳細設定」からアクセス
